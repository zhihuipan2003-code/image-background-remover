# Image Background Remover

AI-powered tool to remove backgrounds from images instantly.

## Features

- 🚀 Lightning fast processing
- 🧠 Smart AI edge detection
- 🔒 Secure & private
- 🌍 Global CDN with Cloudflare
- 📱 Mobile responsive

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

Automatically deployed to Cloudflare Pages on every push to `main` branch.

**Live URL**: https://imagebackgroundremover.pics

## Tech Stack

- Vite - Build tool
- Cloudflare Pages - Hosting & CDN
- GitHub - Version control
- AI/ML - Background removal (coming soon)
